# bun-ts-api

To install dependencies:

```bash
bun install
```
Server:

To run:
```bash
bun run start
```
To create a post:
```bash
{"title": "Post title", "content": "Post content"}
```
URL
```bash
http://localhost:3000/api/posts/'id'
```

Client:

```bash
Open index.html
```

Database:
```bash
Can use any relational database. Currently Postgres is being used.
```
