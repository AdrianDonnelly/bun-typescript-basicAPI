# bun-ts-api

To install dependencies:

```bash
bun install
```
Server:

To run:
```bash
bun run start
```
To create a post:
```bash
{"title": "Post title", "content": "Post content"}
```
URL
```bash
http://localhost:3000/api/posts/'id'
```

Client:
To start:
```bash
Open index.html
```

Database: Postgres
To start:
```bash
sudo service postgresql start
```
